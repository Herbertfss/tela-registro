# Tela de Registro Profissional

[![Status](https://img.shields.io/badge/status-em%20desenvolvimento-yellow)]()

## 📋 Sobre o Projeto

Tela de registro moderna, responsiva e acessível desenvolvida com HTML5, CSS3 e JavaScript puro. O projeto foca em proporcionar uma experiência de usuário excepcional com validações em tempo real e feedback visual imediato.

## ✨ Funcionalidades

- ✅ Design responsivo (mobile-first)
- ✅ Validação de formulário em tempo real
- ✅ Medidor de força de senha
- ✅ Mostrar/ocultar senha
- ✅ Formatação automática de telefone
- ✅ Feedback visual com toast notifications
- ✅ Acessibilidade (navegação por teclado, ARIA labels)
- ✅ Código semântico e comentado

## 🛠️ Tecnologias Utilizadas

- **HTML5** - Estrutura semântica e acessível
- **CSS3** - Flexbox, Grid, variáveis CSS, animações
- **JavaScript (ES6+)** - Validação, manipulação do DOM, classes

## 🚀 Como Executar

1. Clone o repositório:
```bash
git clone https://github.com/Herbertfss/tela-registro.git
```
2. Abra o arquivo index.html no seu navegador
